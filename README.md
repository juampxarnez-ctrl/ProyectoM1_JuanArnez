# Generador de Paletas de Colores
*Aplicación web interactiva que permite generar paletas de colores aleatorias de forma dinámica. El usuario puede elegir la cantidad de colores y el formato de visualización (HEX o HSL), el cual lo facilita a explorar, combinar colores y tener mayor variedad a la hora de elegir el mismo*

### Funcionalidades principales:

* **Generación de paletas de colores aleatorias**
* **Selección del tamaño de la paleta (6, 8 o 9 colores)**
* **Cambio de formato de color (HEX o HSL)**
* **Visualización de cada color junto a su código**
* **Copiado de color al hacer click**

### Cómo usar:

* **Clonar el repositorio**
* **git clone https://github.com/juampxarnez-ctrl/ProyectoM1_JuanArnez.git**
* **Abrir el archivo index.html en el navegador**
* **Seleccionar tamaño de la paleta**
* **Formato de color (HEX o HSL)**
* **Hacer click en "Generar Paleta"**
* **Clickear cualquier color para copiar su código**
